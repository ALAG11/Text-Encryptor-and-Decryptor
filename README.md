# Text Encryptor and Decryptor 🔐✉️

Welcome to the **Text Encryptor and Decryptor**! This command-line tool, written entirely in Python, allows you to securely encrypt and decrypt text with ease.

## 🚀 Features

- **Robust Encryption**: Encrypt any kind of text using strong encryption algorithms.
- **Seamless Decryption**: Decrypt your text securely with the provided key.
- **Command-Line Interface**: Easy-to-use CLI for quick operations.

## 🛠 Technologies Used

- **Python**: 100% of the project

## 📚 Getting Started

Follow these steps to get the Text Encryptor and Decryptor tool up and running:

1. **Clone the repository**:
    ```bash
    git clone https://github.com/ALAG11/Text-Encryptor-and-Decryptor.git
    ```
2. **Navigate to the project directory**:
    ```bash
    cd Text-Encryptor-and-Decryptor
    ```
3. **Install required packages**:
    ```bash
    pip install -r requirements.txt
    ```
4. **Run the application**:
    ```bash
    python Text_Encryptor_And_Decryptor.py
    ```
